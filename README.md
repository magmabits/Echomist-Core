This mod expands upon the Deep Dark by adding new uses for Sculk!

### Features
- Sculk Bricks (Including Stairs, Slabs and Walls)
- Sculk Tiles (Including Stairs, Slabs and Walls)
- Chiseled Sculk
- Sculk Pillars
- Potions of Silence
  - Can be brewed using a Sculk Sensor
  - Mutes all sounds made by an entity (walking, running, eating,...)

Echo Shards can now be used as armor trim materials!

---
### About Echomist

Echomist is an ongoing, work-in-progress mod series, with this just being a standalone mod!
More is coming soon, development progress can be viewed on the [Discord](https://discord.gg/gRFRDhFn2p)!

This mod is not required for other mods of the "Echomist"-Series to work!
This mod will not be ported to other mod loaders or versions, unless I am willing to do so.
