# Echomist: Core

This mod tries to expand the Deep Dark by adding new building blocks, a new potion and a new trim material!

**Echomist: Core adds the following:**
- Sculk Bricks (Including Stairs, Slabs and Walls)
- Sculk Tiles (Including Stairs, Slabs and Walls)
- Chiseled Sculk
- Sculk Pillars
- Potions of Silence
  - Can be brewed using a Sculk Sensor
  - Mutes all sounds made by an entity (walking, running, eating,...)
- Echo Shards are now valid armor trim materials

---
Echomist is an ongoing, work-in-progress mod series, with this just being a standalone mod!
